# Keyboard Navigation Plugin for Better Battlelog

## Assignment and Dogtag List

| Keys    | Actions                        |
|---------|--------------------------------|
| W or ↑  | Navigation Up                  |
| S or ↓  | Navigation Down                |
| A or ←  | Navigation Left                |
| D or →  | Navigation Right               |
| Enter   | Equip Dogtag, Track Assignment |
